Sirius

Sirius was the first enterprise server I ever got ahold of. It runs a lot of what I host!

Specs

  • Model: Dell PowerEdge R610
  • CPU: 2x Intel Xeon E5620 (16) @ 1.615GHz
  • RAM: 96GB DDR3 ECC
  • Storage: 1x 1TB 7.2K SAS
  • OS: Ubuntu 22

Neofetch output of Sirius.

Services

  • This very website!
  • LOTS of other random little webhosting things, such as:
    • RPC - Apache Guacamole
    • Password Manager - Vaultwarden
    • Monitoring - Grafana
    • and more!

Notes

  • Sirius is named after the star Sirius, the brightest star in the night sky!
  • This server was originally named "Microwave" on account of the noise it produces. Turns out, servers can get louder. MUCH louder. Thank you for teaching me that, Antares!
