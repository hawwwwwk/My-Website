Hanzi Picture

Hiya!

Hanzi Picture is a project I have been slowly chipping at for a while, and it's still far from completion.

The goal of this project is to have an image -> ascii art converter, but using Hanzi/Kana characters... something like this:

My original project was written in Java (for some reason), and then converted to Python, and now will be converted into something that can be used on a webpage for accessibility (so the user doesn't need to build/install anything...)
https://github.com/hawwwwwk/img-hanzi




The original project would use "stroke weight", determinant on the brightness of the pixels it was copying. I'm hoping that I can have the program look for edges and try to draw based on that, using the curvature of the characters to draw shapes (like in the example picture!)

The scope of this project makes me worried that I'll never finish it, but I've been slowly pushing.
I will probably do some sort of glyph profiling per character, and match that to a grid on the image. It's hard to explain what I want out of this project, but I can imagine it. It's just been difficult to get it translated to code.
